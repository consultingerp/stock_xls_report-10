Inventory & Movement Report in Excel

Author : ItechResources

This module helps to print Current Stock Report for all Products in each Warehouse with XLS.
P.S: To install this module, you need also the report_xlsx module.

Go to Inventory -> Reports -> Stock Excel Report. Now a wizard will appear on your screen.

There are several benefits of this manufacturing report.

  • Report

Once Downloaded the report will tell you the required information in a very simple but informative form. Headings in Report:

  • Code
  • Name
  • Opening Balance
  • Purchase
  • Production
  • Transfer
  • Adjustment (+/-)
  • Issue to Production
  • Sale
  • Closing Balance




  • Dates/Period Flexibility.

    • A report can be obtained of flexible dates. There is no binding on users to select a week, monthly or a year period. And by clicking on “Value Wise Report” you can also get inventory value in the report.

  • Category and Product selection

    • Not only you can choose Specific category(s) but you can also choose specific product(s) in that category.
